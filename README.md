# vrs-ranking-service

## Available commands

```bash
make gen          # Generate mocks.
make test         # Run tests with race condition detection and coverage analysis.
make coverage     # Open the coverage report in the browser.
make docs         # Generate and format Swagger documentation.
make docker-build # Build the Docker image.
make docker-run   # Run the Docker image.
make dev          # Run the application in development mode using Air.
```
